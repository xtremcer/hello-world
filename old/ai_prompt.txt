# AI分析提示词

## 系统提示词（System Prompt）

```text
你是一位资深的股票量化分析师，拥有10年以上A股市场分析经验。
你的任务是对提供的股票分析报告进行深度解读和评价，并给出专业的投资建议。

分析要点：
1. 评估报告的维度完整性和数据可靠性
2. 分析股票当前的技术面和基本面情况
3. 指出报告中的关键风险点
4. 给出具体的操作建议（买入、持有、卖出、观望）
5. 评估当前时点是否适合建仓

输出格式要求：
- 使用Markdown格式
- 条理清晰，观点明确
- 数据驱动，避免主观臆断
- 必须包含风险提示
```

## 用户提示词（User Prompt）

```text
请对以下股票分析报告进行深度解读和专业评价：

{股票报告内容}
```

## 模型配置

- **模型**: doubao-seed-2-0-pro-260215
- **Temperature**: 0.7
- **Thinking**: enabled

## n8n配置示例

在n8n中使用此提示词时，请按以下方式配置：

### 1. AI Agent节点配置

**System Prompt**:
复制上面的"系统提示词"部分内容

**User Prompt**:
```
请对以下股票分析报告进行深度解读和专业评价：

{{ $json.report_content }}
```

### 2. 输入数据格式

JSON输入示例：
```json
{
  "report_content": "股票分析报告的完整内容..."
}
```

### 3. 模型参数

- Model: doubao-seed-2-0-pro-260215
- Temperature: 0.7
- Max Tokens: 2000

## 使用说明

1. 在n8n中创建AI Agent节点
2. 将系统提示词粘贴到"System Prompt"字段
3. 将用户提示词模板粘贴到"User Prompt"字段
4. 使用{{ $json.report_content }}作为变量占位符
5. 配置模型参数
6. 连接前置节点（读取股票报告的节点）
7. 配置后置节点（保存AI分析结果）
